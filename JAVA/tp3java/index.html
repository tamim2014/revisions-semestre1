Conclusion: tableau et variable static

Un tableau c'est juste une liste. Il se déclare et s'instancie comme une classe :
  •   int   monTableau[ ] =  new   int[10];     un tableau de 10 lignes
  •   int[ ]   monTableau =  new   int[11];     un tableau de 11 lignes
Les variables ou méthodes static appartiennent à la classe elle-même.
On peut donc les utiliser sans avoir une instance créée
