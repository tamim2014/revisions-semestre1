Conclusion: tableau et variable static

Un tableau c'est juste une liste. Il se déclare et s'instancie comme une classe. Pour ce faire il éxiste 2 syntaxes différentes.

Exemples syntaxe1 (Affectation du nom de la variable) :
  •   int   monTableau[ ] =  new   int[10];     //un tableau de 10 lignes
  •   int   monTableau[][] =  new   int[10][2];   //10 lignes et 2 colonnes


Exemples syntaxe2 (Affectation du type de données) :
  •   int[ ]   monTableau =  new   int[11];     //un tableau de 11 lignes
  •   int[][]  monTableau =  new   int[11][3];   //11 lignes et 3 colonnes

Il éxiste des variables et des méthodes qui appartiennent à la classe elle-même.
On les dit : static
On peutes utiliser directement: sans aucun objet d'instance
