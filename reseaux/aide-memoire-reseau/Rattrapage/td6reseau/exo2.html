• Plage par l'adresse Host

Exo2

On donne: L'@Host, et le masque
On demande: L'@Fast, et l'@Last

  1. @Host:23.25.68.2/255.255.224.0

    R=8, S=11  
    @R: 25.68.2 ⇒ 25.010|00000.0 = 25.64.0
    @B: 25.68.2 ⇒ 25.010|11111.2555=25.95.255
     @Fast = 23.25.64.1
    @Last = 23.25.95.254

  2. @Host:198.53.64.7/24

    R=8, S=? ( Pas d'coupure !)
    @R: 7 ⇒ 0
    @B: 7 ⇒ 255

     @Fast = 198.53.64.1
    @Last = 198.53.64.254

  3. @Host:131.107.56.25/21

    R=16, S=5  
    @R: 56.25 ⇒ 00111|000.0 = 56.0
    @B: 56.25 ⇒ 00111|111.255=63.255

     @Fast = 131.107.56.1
    @Last = 131.107.63.254

  4. @Host:148.53.66.7/20

    R=16, S=4  
    @R: 66.7 ⇒ 0100|0000.0 = 64.0
    @B: 66.7 ⇒ 0100|1111.255 = 79.255

     @Fast = 148.53.64.1
    @Last = 148.53.79.254

  5. @Host:1.1.0.1/255.255.0.0

    R=8, S=8(pas d'coupure !!)  
    @R: 1.0.1 ⇒ 1.0.0
    @B: 1.0.1 ⇒ 1.255.255

     @Fast = 1.1.0.1
    @Last = 1.1.255.254

Conlusion :
  Dans tous les cas il faut :
  • Une @Réseaux   ⇒   R
  • Et un Masque     ⇒   S
  pour determiner une plage d'adressage
