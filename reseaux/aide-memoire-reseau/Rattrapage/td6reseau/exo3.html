Exo3

Une societé possède 73 machines qu'elle souhaite repartir en 3 sous-réseaux:

  • SR1: 21 Machines(Département com)
  • SR2: 29 Machines(Département R&D)
  • SR3: 23 Machines(Direction)

  1. Choisir la classe optimale

    Classe C: Nb @ = 28-2 = 254 h
    254 hosts > 29 hosts
    La classe C  est optimale 

  2. Identifiant sous-réseaux ?

    Nb SR = 2S - 2 ( si RFC 950)
    3 ≤ 23-2
    S = 3bits

  3. Masque de sous-réseaux ?

    Classe C     ⇒ R = 24 bits
    3 sous-réseaux ⇒ S = 3  bits
    Masque = /27  
    Masque=255.255.255.224

  4. Nb de machines par sous-réseau

    Masque = /27   ⇒ H = 5  bits
    Nb @ = 25  - 2 
    Nb @ = 30 machines/SR

Pré-réquis Question5 :
    Pour les machines réellement installées :
  • @Fast = @R + 1
  • @Last = Nb @
    Pour les machines possibles :
  • @Fast = idem
  • @Last = @B - 1

    L'@R donne R
    Le masque donne S et @B
