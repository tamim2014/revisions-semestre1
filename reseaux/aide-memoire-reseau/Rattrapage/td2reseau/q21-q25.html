  1. Quel matériau est utilisé pour le média de réseau?
     A• fil
     B• câble
     C• fibre optique
     D• toutes ces réponses

    C• fibre optique

  2. Laquelle de ces unités n'est PAS une unité de couche1?
     A• Concentrateur
     B• Répéteur
     C• Commutateur
     D• Émetteur-récepteur

    C• Commutateur

  3. Si quatre hôtes sont connectés à un concentrateur, puis à Internet, combien faut-il d'@IP
    pour ces cinq unités?
     A• Un
     B• Deux
     C• Quatre
     D• Cinq

    C• Commutateur

  4. Commutateurs:

     A• Couche 2 ; utilise les @IP
     B• Couche 3 ; utilise les @IP
     C• Couche 2 ; utilise les @MAC
     D• Couche 3 ; utilise les @MAC

    C• Couche 2; utilise les @MAC

  5. Les unités d'interconnexion de réseaux sont utilisées
    pour relier des
     A• LAN/LAN
     B• LAN/WAN
     C• WAN/WAN
     D• toutes ces réponses

    D• toutes ces réponses
