8.1 INTERFACE

Une classe implemantant une interface, doit implementer toutes les mthds de l'interface

8.2 CLASSE ABSTRAITE

Une classe abstraite n'est pas obligé d'implementer toutes les mthds de l'interface

Lorsqu’une classe hérite d’une classe abstraite,
elle doit :
  1. Soit implémenter toutes les méthodes abstraites de sa classe mère en les dotant d’un corps ;
  2. Soit être elle-même abstraite ( si au moins une des méthodes abstraites de sa classe mère reste abstraite).

Une classe abstraite n’a pas de constructeur
( donc elle ne peut être instanciée)
