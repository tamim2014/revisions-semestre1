Conclusion: La méthode toString()

Elle est définie dans la classe Object ; en conséquence toutes les classes Java en hérite.
Elle ne fait pas grand-chose :
• Elle renvoie juste le nom de la classe de l'objet concerné [suivi de l'adresse de cet objet ].
• Lorsqu'on définit une classe, il est très utile de redéfinir la méthode toString() afin de donner une description satisfaisante des objets de cette classe
