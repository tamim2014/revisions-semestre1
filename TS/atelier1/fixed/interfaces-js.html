Interface HTMLElement

C'est une interface abstraite pour le DOM.
Elle est heritée par tous les éléments HTML

⏩  Propriétés

  • element.id
  • element.lang
  • element.title

  • element.className
  • element.classList  ⚠

L'utilisation de classList est une alternative à la propriété element.className qui renvoie une chaine composée de la liste des classes, séparées par des espaces.

Elle utilise 6 méthodes:
   ► element.className.add( String [, String] )   ⚠
   ► element.className.remove( String [, String] )
   ► element.className.toggle( String [, force] )
   ► element.className.item( Number )
   ► element.className.contains( String )
   ► element.className.replace( oldClass, newClass )
Plus de détails sur: developer.mozilla.org
