• C'est quoi une extension(plugins)?

  • Module : Pour ajouter des fonctionnalités à wp
    • Pour la partie visible du site(front office)
    • Pour la partie administration Wordpress(back office)
  • Extension = dossiers composés de fichiers PHP

  • Exemple: Plugin TinyMCE Widget(mise en page WYSIWYG)
  • Exemple: Plugin MailChimp( ajout de newsletter )
  • Exemple: Plugin PolyLang( ajout d'un traducteur )
  • Exemple: Plugin WooCommerce(catalogue produits, e-commerce)
  • Exemple: Plugin WP Accessibility
  • Exemple: Plugin Accessible Poetry(ENG)
  • Exemple: Plugin wA11y - The web Accessibility Toolbox
