•Que font les fonctions suivantes

  1. wp_head() ?            Ajout de style CSS ou de script js functions.php
  2. wp_footer() ?          Ajout de style CSS ou de script js functions.php
  3. wp_enqueue_style($clé, $valeur) ?          Ajout de fichier CSS functions.php
  4. register_nav_menu($identifiantMenu, $description)?   Enregistrement du menu functions.php
  5. wp_nav_menu(array('zone' =>'identifiantMenu'))?   header.php Affiche le menu dans l'ADMINISTRATION
  6. have_posts()?   index.php Teste s'il ∋ des articles
  7. the_posts()?     index.php Accès en lecture à la BD: stockage infos des articles sur une variable globale

•Comment implementer un menu

  • register_nav_menu() :    function.phpDéfinition(enregistrement) du menu
  • wp_nav_menu() :    header.phpAppel(Affichage) du menu dans la zone d'administration
  • Avant d'appler wp_nav_menu() dans header.php, il faut le définir dans: functions.php

•Comment implementer une barre latérales

  • register_sidebar( array('id' => 'barre_laterale') ) :    function.phpDéfinition(enregistrement) de la barre latérale
  • dynamic_sidebar('barre_laterale') :    index.phpAppel(Affichage) de la barre latérale dans la zone d'administration
  • Avant d'appler dynamic_sidebar() dans index.php, il faut le définir dans: functions.php

•Comment implémenter un arrière-plan personnalisé(custom backgrounds)

  • add_theme_support( 'custom-background', $defaults ) :    function.phpDéfinition(enregistrement) de la couleur d'arrière-plan
  • body_class() :    header.phpAppel(Affichage) de l'arrière-plan dans la zone d'administration
  • Avant d'appler body_class() dans header.php, il faut le définir dans: functions.php

•Comment implémenter un en-tête personnalisé(custom header)

  • add_theme_support( 'custom-header' ) :    function.phpDéfinition(enregistrement) de l'image d'en-tête
  • header_image() :    header.phpAppel(Affichage) de l'image d'entête dans la zone d'administration
  • Avant d'appler body_class() dans header.php, il faut le définir dans: functions.php

•Comment implémenter un logo(custom logo)

  • add_theme_support( 'custom-logo' ) :    function.phpDéfinition(enregistrement) du logo
  • the_custom_logo() :    header.phpAppel(Affichage) du logo dans la zone d'administration
  • Avant d'appler body_class() dans header.php, il faut le définir dans: functions.php

•À quoi sert les hooks?

Accroche les fonctions au core de WordPress

•Quels sont les 2 type de hooks?

  • hooks action : add_action($nomduhook, $fonction) Pour écrire vos propres foncion | ne renvois pas de valeur
  • hooks filtre : add_filter($nomduhook, $fonction) Modif f° qui ∋ | renvois une valeur

•Donner 2 exemples de hooks d'action

  • wp_head() :    header.phpinsertion de code (css, javascript) dans <head> ... </head>
  • wp_footer() :    footer.php...
  • Avant d'appler ces fonctions il faut les definir dans: functions.php
  • Ainsi on met le style ou le script sans recours à une feuille externe

•Inserer le style "template.css" dans votre thème ( usage d'une feuille externe)

  • wp_enqueue_style($clé, $valeur)
  • wp_enqueue_style("template", get_template_directory_ury()."css/template.css")
  • get_template_directory_ury()?  renvoie le repertoire du thème(wp-content/thème/masterhandi)
