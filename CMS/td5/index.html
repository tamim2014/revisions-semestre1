Conclusion: Notion de hooks

Personaliser le fonctionnement d'un logiciel, en lui faisant réaliser des actions suplémentaires.
Ce sont des points d'entrée vers des listes d'actions.
On distingue 2 types de hooks chez WordPress :
• Les hooks d'action: pour ajouter de nouvelles fonctionnalités [les actions sont basées sur des évennements ]
• Les filter: pour modifier une donnée construite par WordPress [ avant de l'afficher ]

Les hooks d'action Description
init Add menu
widget_init Add sidebar
wp_head
wp_footer
admin_enqueue_scripts
publish_post Ajoute des fonctionnalités exécutées à chq publicatiob d'un article
